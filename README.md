# E-Commerce Sales & Shipping Dashboard (Excel)

## Overview
This project is an interactive E-Commerce Sales & Shipping Dashboard developed in Microsoft Excel to analyze sales performance, shipping efficiency, customer trends, and product insights.

The dashboard provides dynamic visualizations and business insights using Excel Power Query, Data Modeling, Pivot Tables, and Pivot Charts.

---

## Tools & Technologies
- Microsoft Excel
- Power Query
- Excel Data Modeling
- Pivot Tables
- Pivot Charts
- Slicers & Interactive Filters
- Dashboard Visualization

---

## Key Features
- Sales Performance Analysis
- Shipping Performance Tracking
- Product Trend Analysis
- Customer Sales Insights
- Region-wise Sales Visualization
- Average Shipping Days Analysis
- Interactive Dashboard with Dynamic Filters

---

## Dashboard Insights
The dashboard includes:
- Net Sales by Category
- Quarterly Product Sales Trends
- Top Customers Analysis
- Shipping Mode Performance
- Average Shipping Days
- Salesperson-wise Performance
- State-wise Sales Visualization

---

## Process Followed
1. Imported raw e-commerce sales data into Excel
2. Cleaned and transformed data using Power Query
3. Built data relationships using Excel Data Model
4. Created Pivot Tables for KPI calculations
5. Developed Pivot Charts for visualization
6. Designed an interactive dashboard using slicers and charts
